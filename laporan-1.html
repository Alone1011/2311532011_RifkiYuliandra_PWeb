Membangun Aplikasi CRUD Sederhana dengan PHP OOP

25 Juni 2024 8 min read

Pendahuluan

Aplikasi CRUD (Create, Read, Update, Delete) merupakan dasar dalam membangun sistem informasi. Praktikum ini bertujuan mengimplementasikan konsep OOP dalam PHP menggunakan database MySQL untuk mengelola data mahasiswa.

Alat dan Teknologi

  • XAMPP – Menyediakan server lokal (Apache & MySQL)
  • Visual Studio Code – Code editor ringan dan fleksibel
  • Git & GitHub – Kontrol versi dan publikasi kode
  • Bootstrap 5 – Untuk antarmuka responsif

Struktur Proyek

crud/
├── assets/
│   ├── css/
│   └── js/
├── config/
│   ├── Config.php
│   └── Database.php
├── function/
│   ├── Alert.php
│   └── Mahasiswa.php
├── model/
│   └── Mahasiswa.php
├── create.php
├── edit.php
└── index.php

Langkah-Langkah Implementasi

1. Buat database pemrograman_web dan tabel mahasiswa dengan kolom id, nim, nama, jurusan.

2. Konfigurasikan koneksi database pada file Database.php: 

class Database {
  private $host = "localhost";
  private $db_name = "pemrograman_web";
  private $username = "root";
  private $password = "";
  public $conn;

  public function getConnection() {
    $this->conn = new mysqli($this->host, $this->username, $this->password, $this->db_name);
    return $this->conn;
  }
}

3. Tambahkan konstanta BASE_URL di file Config.php.

4. Buat file Alert.php untuk menampilkan notifikasi operasi data.

5. Bangun class Mahasiswa di folder model/ dengan method: create(), read(), update(), dan delete().

6. Gunakan function/Mahasiswa.php sebagai controller penghubung antar model dan view.

7. Buat tampilan di index.php, create.php, dan edit.php.

Fitur Tambahan

  • Validasi sederhana pada form input
  • Struktur file modular dan terorganisir
  • Penggunaan $_SESSION untuk pesan sukses atau gagal
  • Desain responsif menggunakan Bootstrap
